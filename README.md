# BERT PEFT Sentiment Analysis

## Overview

This project compares different Parameter-Efficient Fine-Tuning (PEFT) techniques for BERT on the SST-2 sentiment analysis dataset.

The objective is to evaluate modern fine-tuning methods such as:

- Full Fine-Tuning
- LoRA
- QLoRA
- AdapterFusion

while comparing their:

- Accuracy
- Training Loss
- Validation Loss
- GPU Memory Usage
- Training Efficiency

---

## Technologies

- Python
- PyTorch
- Hugging Face Transformers
- PEFT
- LoRA
- QLoRA
- AdapterFusion
- Weights & Biases
